# 1.查看端口是否被占用
```shell
# 查看8081口是否被占用
sudo lsof -i :8081

netstat -anp tcp | grep 8081

nc -w 10 -n -z 127.0.0.1 8070-8090

# 直接结束占用端口的所有进程：
lsof -P | grep ':80' | awk '{print $2}' | xargs kill -9
```


# 2.给当前用户添加免密sudo权限
```shell

sudo visudo

# 到文件尾的root and users添加一行，后保存重启终端
username      ALL = (ALL) NOPASSWD:ALL
```
